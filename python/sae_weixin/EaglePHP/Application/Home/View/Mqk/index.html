诸葛武侯 未来预知术，相传为蜀汉诸葛武侯著，邵康节演。又称之为《蜀汉诸葛武侯马前课》，原藏于一八八一年法兰西巴黎图书馆。曾被列为秘籍。原占法：凡占此卦，须于清晨洗净后，选铜钱八枚或铜元亦可，取其中一枚以红线系之，焚香净手，虔诚默告一通，随将钱含掌中，两手高举向空，速摇数下，摇毕，将钱按照《占卦式》八卦图上的次序依次排列，此时看系红线者排在某卦上，即为某卦。两次成卦，第三次求动爻。占法较为复杂，专业人士才懂使用。为了方便大家求占，简化了程序，心念动处即可求占。
